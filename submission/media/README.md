## Demo Note

For the strongest recording, run the public Vercel UI with the local FastAPI
bridge at `http://127.0.0.1:8000`. This lets the demo show Obsidian note
writing, RAG retrieval, and SQLite learner memory while keeping private files
and API keys local.
